# dotfiles

MacOS用の初期セットアップを行います。

## 管理対象ファイル

- `~/.config/git/ignore` - Gitのグローバル除外設定
- `~/.config/mise/config.toml` - miseの設定
- `~/.config/zed/keymap.json` - Zedエディタのキーマップ設定
- `~/.config/zed/settings.json` - Zedエディタの一般設定
- `~/.gitconfig` - Gitの設定
- `~/.vimrc` - Vimの設定
- `~/.zshrc` - Zshの設定
- `~/.zprofile` - Zshの環境変数等の設定
